Ein modernes Miet- und Verkaufs-Plugin mit Stripe-Integration, Produktvarianten, Extras, Versandoptionen, Kundenkonto, PDF-Rechnungen und vielem mehr – ideal für Vermietung und Verkauf von Partyzubehör, Event-Equipment oder anderen Produkten.

== Description ==

Dieses Plugin bietet eine komplette Miet- und Verkaufsverwaltung direkt im WordPress-Backend:

* Produktverwaltung mit Varianten, Extras und Lagerbestand
* Stripe Checkout (Hosted & Embedded) mit Unterstützung für Abos, PayPal via PMC und Zusatzartikel
* Kundenkonto mit Bestellhistorie, Rechnungsdownload, Zahlungsmethodenverwaltung und Rückgabebestätigung
* Admin-Dashboard mit Bestellungen, Kunden, Umsatzstatistiken und Rückgabestatus
* PDF-Rechnungen inkl. enthaltenem Umsatzsteueranteil
* Content-Blöcke für Produktlisten
* Globale Versandkostenverwaltung
* Kompatibel mit Freemius für Lizenzen und Updates

**Hinweis:** Dieses Plugin ist für die Vermietung/Vermarktung beliebiger physischer Produkte geeignet, nicht jedoch für Immobilien oder Fahrzeuge.

== Installation ==

1. Laden Sie den Plugin-Ordner in das Verzeichnis `/wp-content/plugins/` hoch oder installieren Sie es über „Plugins → Installieren“ im WordPress-Backend.
2. Aktivieren Sie das Plugin über „Plugins → Installierte Plugins“.
3. Konfigurieren Sie das Plugin über den neuen Menüpunkt im WordPress-Backend (Stripe-Keys, Versandoptionen, Produkte, etc.).

== Frequently Asked Questions ==

= Ist Stripe zwingend erforderlich? =
Ja, aktuell ist Stripe (inkl. PMC für PayPal) die primäre Zahlungsabwicklung.

= Kann ich das Plugin ohne Freemius nutzen? =
Ja, aber Updates und Lizenzverwaltung funktionieren nur mit aktiver Freemius-Integration.

= Unterstützt das Plugin auch Käufe statt Miete? =
Ja, es kann zwischen Miet- und Verkaufslogik umgeschaltet werden.
